## Keyboard Shortcuts

- `Ctrl+T` - New tab
- `Ctrl+W` - Close tab
- `Ctrl+Tab` - Next tab
- `Ctrl+Shift+Tab` - Previous tab
- `Ctrl+C` - Interrupt
- `Ctrl+L` - Clear screen
- `Up/Down` - History navigation
- `Tab` - Path completion
